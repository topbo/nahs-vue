# nahs-vue
## Nutrition and Health Score(营养健康评分平台)
### 项目背景
 现如今人们生活水平提高，越来越多人的希望健康饮食，却没有一个能提供标准的平台，使人能够对一天的饮食情况作出正确地判断。希望自己能开发这样的平台，已评分及推荐的形式让大家健康饮食，快乐生活。
### 项目搭建
* Vue-cli 搭建项目框架
* vue-router 构建前端路由
* axios 异步请求数据
* node express框架 + mongoose 构建数据api
* less CSS 预处理器
* WebPack 打包
### 项目运行
* 下载项目 到项目路径下
* `npm install` //安装依赖
* `node app` //运行数据api
* `npm run dev` //运行项目
